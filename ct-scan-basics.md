# CT Scan Basics

## Definition
Computed Tomography (CT) uses X-rays to create cross-sectional images of the body.

## Key Features
- Multiple X-ray beams
- Computer-processed images
- High detail resolution

## Uses
- Brain imaging
- Trauma assessment
- Tumor detection

## Advantages
- Detailed internal view
- Faster than MRI

## Limitations
- Higher radiation dose than X-ray
